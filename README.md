# yaai_cloud - ABAP AI tools Cloud - SAP Fiori Elements Application
This App is an example of how to integrate ABAP AI tool Cloud capabilities into SAP Fiori Elements Applications.

The current focus of this documentation is detailing the integration of a simple chat user interface into a SAP Fiori Elements Application.

Because SAPUI5 does not include dedicated controls to build a chat interface, a custom solution is required. This documentation provides the necessary steps for implementing this custom chat UI within your SAP Fiori Elements Application.
